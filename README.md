# Remake Pixel — Static Site

Upload this folder to Vercel as **Static Site**.
No build needed — files are already compiled.

## Vercel Deploy Steps
1. Go to vercel.com → New Project
2. Choose "Import from GitHub" or "Upload"
3. Framework Preset: **Other**
4. Deploy
5. Add custom domain: remakepix.com
